# Project Roadmap

This document provides an overview of the planned development phases and milestones for the AWS Clinical Support Copilot project.

## Milestones

- **Day 1-2:** Initial project setup, requirements gathering, and infrastructure planning.
- **Day 3-5:** Core Lambda function development and integration with AWS services.
- **Day 6-8:** User interface prototyping and feedback collection.
- **Day 9-12:** Advanced feature implementation and security hardening.
- **Day 13-14:** Testing, documentation, and deployment preparation.

## Folder Structure
- `Docs/` — Documentation and design notes
- `Infra/` — Infrastructure as code (IaC) scripts
- `Lambdas/` — AWS Lambda functions
- `Prompts/` — Prompt engineering and AI prompt files
- `Roadmap/` — Daily breakdowns and progress tracking
- `Samples/` — Example data and sample files
- `Test-UI/` — User interface prototypes and test harnesses

## Notes
- See the `Roadmap/` folder for detailed daily plans.
- This file should be updated as the project evolves.
